Collaborate



Tools for remixing
Interested in tools that can help you communicate your message? Here is a collection of free and open source software tools for working with CC-licensed content.

Elogio

A browser plugin and API for transfering attribution information of images you're reusing.

View tool »

Open Attribute

A browser plugin that makes it easier to detect a CC license websites you visit, and copy an attribution in a single click.

View tool »

Solvonauts

A search engine that searches for only CC-licensed and public domain learning resources.

View tool »


CC Search

Convenient access to the search functions of repositories that allow searching by CC license.

View tool

SBCTC Attribution Builder

Web form for creating attribution text for content you're reusing.

View tool
